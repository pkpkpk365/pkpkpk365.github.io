# 《别怕，Excel VBA 其实很简单》的学习笔记
## 第3章 学习语法
### 3.2 VBA中的数据及数据类型
- excel里面的数据只有文本、数值、日期值、逻辑值、错误值5种类型
- VBA的数据类型有：布尔型（Boolean）、字节型（Byte）、整数型（Integer）、长整数型（Long）、单精度浮点型（Single）、双精度浮点型（Double）、货币性（Currency）、小数型（Decimal）、字符串型（String）、日期型（Date）、对象型等。
- 正确设置数据类型可以有效减少内存占用，提高运行速度。
### 3.3 常量和变量
- 变量可以随时更换，通常用于存储临时保存的数据和对象
- 常量一旦存入数据，就不能更换
### 3.4 在程序中使用变量存储数据
- 声明变量  
语法格式：Dim 变量名 As 数据类型
```vba
Dim IntCount As Integer  'Dim 变量名 As 数据类型
```
