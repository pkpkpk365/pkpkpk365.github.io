# 个人web页面小工具集合
My personal web tools
## 站点：pkpkpk365.github.io
## 使用方法：
1.新建文件夹（英文命名），然后将小工具放入文件夹。<br>
2.编辑tools.json，将对应的小工具按照对应的格式添加到json文件中即可。<br>
## json文件格式实例：<br>
```json
[
  {
    "name": "股市年化收益率计算器",
    "desc": "输入起始日期、点位和结束日期、点位计算",
    "url": "./stock-annual-return/",
    "category": "金融工具",
    "recommended": true
  },
    {
    "name": "代码收集本",
    "desc": "记录我日常生成和高频使用的各类代码",
    "url": "./codebook/",
    "category": "编程工具",
    "recommended": true
  }
]
```
