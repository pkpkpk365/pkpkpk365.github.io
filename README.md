# 使用方法：
1.新建文件夹（英文命名），然后将小工具放入新建文件夹。<br>
2.编辑tools.json，将对应的小工具按照对应的格式添加到json文件中即可。<br>
3.json文件的编辑格式：<br>
=======================<br>

  {<br>
    "name": "股市年化收益率计算器",<br>
    "desc": "输入起始日期、点位和结束日期、点位计算",<br>
    "url": "./stock-annual-return/",<br>
    "category": "金融工具",<br>
    "recommended": true<br>
  },<br>
    {<br>
    "name": "代码收集本",<br>
    "desc": "记录我日常生成和高频使用的各类代码",<br>
    "url": "./codebook/",<br>
    "category": "编程工具",<br>
    "recommended": true<br>
  }<br>
]<br>
=======================<br>
